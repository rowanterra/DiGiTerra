# Example datasets for DiGiTerra

Common ML datasets used to validate that DiGiTerra works end-to-end. Stored in the repo (on GitHub) only; the app does not bundle them.

All are from [scikit-learn](https://scikit-learn.org/stable/datasets.html) (BSD license), except `synthetic_regression.csv` (generated via `make_regression`). Generated by `scripts/generate_example_datasets.py`.

## How to use with DiGiTerra

1. Run the app (`python app.py` or desktop build).
2. **Upload** the CSV (e.g. `examples/iris.csv`).
3. **Data exploration**: run correlation matrices, pairplots, etc.
4. **Model preprocessing**: choose columns (spreadsheet letters). Set indicators vs targets, then **Continue to Modeling**.
5. **Modeling**: pick the matching task (Classification, Regression, or Clustering) and a model. Train and review metrics and visualizations.

### Suggested column setup

**Classification**

- **iris.csv**: Indicators `A-D`, Target `E` (species).
- **wine.csv**: Indicators `A-M`, Target `N` (class).
- **breast_cancer.csv**: Indicators `A-AD`, Target `AE` (diagnosis).

**Regression**

- **diabetes.csv**: Indicators `A-J`, Target `K` (target).
- **synthetic_regression.csv**: Indicators `A-J`, Target `K` (y).
- **linnerud.csv**: Indicators `A-C` (Chins, Situps, Jumps), Targets `D-F` (Weight, Waist, Pulse) for multi-target regression.

**Clustering**

- **iris_clustering.csv**, **wine_clustering.csv**, **breast_cancer_clustering.csv**: Select all columns as indicators. Do **not** set a target. Choose the Clustering task and run (e.g. K-Means, GMM).
